# Point_Cloud_from_Depth_Camera

###Description:
1. This small project aims at generating PCL point cloud and opencv mat data from some depth cameras.
2. It now supports Asus depth camera, bumblebee2 and kinect2.
3. Adandon what you don't need.

###Dependence:
0. linux environment
1. opencv 3.0 or higher version
2. openni(for asus or kinect1)
3. libfreenect2(for kinect2 use cpu pipeline)
4. triclops and flycapture(for bumblebee2)
5. boost
6. c++11 support
